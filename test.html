About the Effect

The "Burn the Web" bookmarklet creates a realistic fire animation that makes any webpage appear to catch fire and dissolve. It uses HTML5 Canvas for the particle effects and CSS filters to make the content appear to burn away.

"Some people just want to watch the web burn." - Anonymous Developer

How It Works

When triggered, the bookmarklet performs the following actions:

  • Creates an HTML5 Canvas overlay covering the entire viewport
  • Generates 100 fire particles with realistic physics and colors
  • Applies progressive CSS filters to make the page appear to char
  • Fades the entire page to black over 5 seconds
  • Cleans up and removes the canvas when complete

Technical Features

The bookmarklet is optimized for mobile performance with several key features:

  • Frame Throttling: Runs at 30 FPS to conserve battery
  • Particle Recycling: Reuses particle objects for efficiency
  • Gradient Rendering: Uses radial gradients for realistic glow
  • CSS Transitions: Smooth filter effects for the burn animation
