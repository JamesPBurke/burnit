📖 Installation Instructions

Follow these steps to install the bookmarklet on Android Chrome:

1 Copy the Code: Tap the "Copy Bookmarklet Code" button above. The code will be copied to your clipboard.
2 Open Bookmarks: In Chrome, tap the three-dot menu (⋮) in the top-right corner, then tap "Bookmarks".
3 Add New Bookmark: Tap the three-dot menu again and select "Add new bookmark" (or tap the ⭐ icon and then "Edit").
4 Name It: In the "Name" field, type Burn It or any name you prefer.
5 Paste the Code: In the "URL" field, delete any existing content and paste the code you copied in step 1. Make sure it starts with javascript:
6 Save: Tap "Save" or the checkmark icon to save your bookmark.

🚀 How to Use

1 Navigate to any webpage you want to burn.
2 Tap the address bar at the top of Chrome.
3 Type "Burn" and look for your bookmark in the suggestions.
4 Tap the bookmark. Watch the page catch fire and burn away! 🔥
💡 Tip: The effect lasts about 5 seconds. Refresh the page to restore it. The bookmarklet is harmless and doesn't modify the actual website.
