Install on Android Chrome

1 · Tap 📋 Copy code below
2 · Bookmark any page (⋮ → Add to bookmarks)
3 · Open ⋮ → Bookmarks, long-press that bookmark
4 · Tap Edit, clear the URL, paste, save
5 · Visit any page, tap the address bar, type Burn
